# OpenLogixRestAPI

## Overview
This project is an API which lets you add, delete and view a list of users. Each user has an ID, name and age.

## API Endpoints
- View Users :  

  Path: "/users" 
  
  Request Type : GET

- Add Users : 

  Path: "/users" 
  
  Request Type : POST

- Delete User : 

  Path: "/users/{userId}" 

  Request Type : DELETE

## Technologies Used
- Java 8
- Spring Boot
- Maven
